## 🚀 Featured Project

### AI Teams Bot (RAG-based Document Assistant)

An AI assistant integrated with Microsoft Teams that answers user queries using internal company documents.

#### Problem
Employees struggle to find information across scattered PDFs, Excel sheets, and docs. Search is keyword-based and unreliable.

#### Solution
Built a **Retrieval-Augmented Generation (RAG) system** that:
- Understands natural language queries  
- Retrieves relevant document chunks  
- Generates contextual answers using an LLM  
- Returns references to source documents  

---

### 🏗️ Architecture

User → Teams Bot → FastAPI → RAG Pipeline → Gemini → Response

**Pipeline breakdown:**
- Document ingestion (PDF, DOCX, XLSX)
- Chunking & preprocessing
- Embedding generation
- Vector similarity search
- LLM response generation with context
